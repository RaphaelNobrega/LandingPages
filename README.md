
# Landing Pages

Estes projetos fazem parte dos primeiros passos dados e que desenvolvi para estudos e treinamento de HTML, CSS.
Portanto, os mantenho aqui em sua forma original... para sempre poder ver o quanto estou evoluindo meus conhecimentos e recordar que a constância trás resultados magnificos.


## Aprendizados

Nos projetos pude aprender HTML, CSS aplicando o uso de classes, separação do site em partes, padrão BEM (CSS) ajudando na estruturação e leitura do código.

## Stack utilizada

**Front-end:** HTML, CSS
